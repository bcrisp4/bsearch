# Smart meter half-hourly data — project idea

Harwood's app has an "Insights" view with half-hourly usage but no
export. Options: their API (none public), screen-scraping the app
(no), or the industry consumer-data route which supposedly lets you get
your own meter data. Parked until winter — would be genuinely useful to
overlay heating hours vs kWh before/after any heat pump decision
([[heat-pump-research]]).

Feb data point worth remembering: when the meter lost signal for 3 weeks
([[energy-costs-first-year]]) the Insights view just… gapped. So build
nothing that assumes continuity.
